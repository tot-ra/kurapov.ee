(2007 - 2008)

[Elitec.ee](http://elitec.ee/)

- Разрабатывал краткосрочные проекты на кастомной CMS (LAMP stack)
- Разработал модуль календаря для CMS с перетаскиваемым UI в стиле Google Calendar
- Поддерживал систему документооборота RMK.ee (Postgres)

Эстонский рынок, небольшая веб-студия, частично связанная с Medicum

Получил опыт с:
- SVN, Oracle
- PHP Zend Framework, Prototype.js

## Проекты

| Название | Описание |
| --- | --- |
| [RMK](http://www.rmk.ee/) | Внутренняя система документов для Центра управления государственными лесами Эстонии. Как и EIQA, проект сначала делался в ADM Interactive, но когда объём задач вышел за разумные рамки поддержки, разработку передали третьим лицам. PostgreSQL, самописные шаблоны, много легаси-кода, гигабайтные базы, ветвление в CVS. |
| [NSA.ee](http://www.nsa.ee/) | Информационная страница министерства обороны. Перенос сайта с существующего движка на Joomla. Управился примерно за 10 часов: перенос дизайна, обновление меню и контента, разборка с движком. |
| [Silmalaser.ee](http://silmalaser.ee/) | Медицинская компания по коррекции зрения. Модуль регистратуры на Oracle, связанный с MIS (инфосистема Medicum). Сделал PEAR-подобную прослойку бизнес-логики (Model в MVC), пробовал xajax для форм. |
| Календарь | Ajax-модуль календаря, функционально похожий на Google Calendar и Kiko. Использует Scriptaculous для drag-and-drop событий, приглашения по почте, формат iCal для совместимости с MS Outlook и просмотр занятости пользователей. |
| Регистратура [Medicum](http://www.medicum.ee/) | Очень большая Oracle 9 база клиентов, расписаний и врачей. Переделывал устаревшую регистратуру. Две независимые компании оценивали объём работ более чем в 800 тыс. крон. |
| Регистратура | Идейно похожа на регистратуру Medicum, но на MySQL и с одной рабочей временной рамкой вместо заранее сгенерированных свободных промежутков. |
